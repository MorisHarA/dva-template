# dva-template

移动端dva初始化模板，由于dva-cli生成的模板功能比较基础，所以此项目致力于能够开箱即用。

## Features
 - 兼容低版本浏览器（ios8, andriod 4.1）
 - 使用viewport高清方案适配各种移动端页面
 - ant-design-mobile 内置
 - 支持路由按需加载
 - 封装request
 - 更爽的mock数据体验
